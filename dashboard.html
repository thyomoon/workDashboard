Setup instructions
1. Create a GitHub account if you don't have one.
2. Go to github.com/settings/tokens → Generate new token (classic) → check only gist scope → copy the token.
3. Create a private gist at gist.github.com with any content (e.g. filename dashboard.json, content {}).
4. Copy the gist ID from the URL (the long hex string).
5. Paste both here. Token and Gist ID are saved only on this device.
